# دليل الإعداد الكامل - تطبيق مراقبة الطفل

## المرحلة 1: تحضير البيئة

### تثبيت Android Studio

1. اذهب إلى https://developer.android.com/studio
2. حمّل Android Studio
3. اتبع خطوات التثبيت
4. افتح Android Studio وانتظر حتى ينتهي من تحميل المكونات

### تثبيت Android SDK

1. افتح SDK Manager من Android Studio
2. اختر الإصدارات التالية:
   - Android 14 (API 34)
   - Android 13 (API 33)
   - Android 12 (API 31)
3. اضغط "Apply" و "OK"
4. انتظر حتى ينتهي التثبيت

## المرحلة 2: استيراد المشروع

1. افتح Android Studio
2. اختر `File > Open`
3. اختر مجلد `ChildMonitorApp`
4. اضغط `OK`
5. انتظر حتى ينتهي Gradle من البناء (قد يستغرق 5-10 دقائق)

## المرحلة 3: تكوين الخادم

### تحديث عنوان الخادم

1. افتح الملف: `app/src/main/java/com/example/childmonitor/api/ApiClient.kt`
2. ابحث عن السطر: `private val baseUrl = "https://your-server.com/api/trpc"`
3. غيّر `your-server.com` إلى عنوان خادمك الفعلي

### تحديث مفتاح Google Maps

1. افتح الملف: `app/src/main/AndroidManifest.xml`
2. ابحث عن: `android:value="YOUR_GOOGLE_MAPS_API_KEY_HERE"`
3. استبدل `YOUR_GOOGLE_MAPS_API_KEY_HERE` بمفتاحك من Google Cloud Console

## المرحلة 4: البناء والتشغيل

### على محاكي أندرويد

1. اضغط على `AVD Manager` في Android Studio
2. اختر جهاز محاكاة أو أنشئ واحداً جديداً
3. اضغط على زر التشغيل
4. انتظر حتى يبدأ المحاكي
5. اضغط `Run > Run 'app'`

### على جهاز حقيقي

1. فعّل وضع المطور على جهازك:
   - الإعدادات > حول الهاتف
   - اضغط على "رقم الإصدار" 7 مرات
2. اختر الإعدادات > خيارات المطور
3. فعّل "تصحيح USB"
4. وصّل جهازك بالكمبيوتر
5. اضغط `Run > Run 'app'`

## المرحلة 5: بناء ملف APK

### بناء APK للإصدار

1. افتح Terminal في Android Studio
2. نفّذ الأمر:
```bash
./gradlew assembleRelease
```

3. سيتم حفظ الملف في:
```
app/release/app-release.apk
```

### بناء APK للاختبار

```bash
./gradlew assembleDebug
```

الملف سيكون في:
```
app/debug/app-debug.apk
```

## المرحلة 6: تثبيت على الهاتف

### الطريقة 1: عبر USB

1. وصّل الهاتف بالكمبيوتر
2. افتح Terminal
3. نفّذ:
```bash
adb install app/release/app-release.apk
```

### الطريقة 2: نقل الملف يدوياً

1. انسخ ملف APK إلى الهاتف
2. افتح مدير الملفات
3. اضغط على ملف APK
4. اضغط "تثبيت"

## المرحلة 7: الإعدادات الأولية

### على هاتف ولي الأمر

1. افتح التطبيق
2. اختر "دخول ولي الأمر"
3. أدخل بريدك الإلكتروني وكلمة المرور
4. اضغط "دخول"

### على هاتف الطفل

1. افتح التطبيق
2. اختر "دخول الطفل"
3. أدخل كلمة المرور التي أنشأتها من حساب ولي الأمر
4. اضغط "دخول"

## المرحلة 8: تفعيل الصلاحيات

### على هاتف الطفل

1. الإعدادات > التطبيقات > تطبيق مراقبة الطفل
2. الصلاحيات:
   - الموقع: اختر "السماح دائماً"
   - المكالمات: اختر "السماح"
   - الرسائل: اختر "السماح"
   - الكاميرا: اختر "السماح"
   - المايكروفون: اختر "السماح"

3. الإعدادات > إمكانية الوصول
   - فعّل "تطبيق مراقبة الطفل"

4. الإعدادات > الأمان
   - فعّل "تطبيق مراقبة الطفل" كمسؤول جهاز

5. الإعدادات > البطارية
   - اختر "تطبيق مراقبة الطفل"
   - اختر "غير محدود"

## استكشاف الأخطاء الشائعة

### خطأ: "Gradle build failed"
```bash
./gradlew clean
./gradlew build
```

### خطأ: "SDK not found"
- تأكد من تثبيت Android SDK
- افتح SDK Manager وثبّت الإصدارات المطلوبة

### خطأ: "Device not found"
```bash
adb devices
adb kill-server
adb start-server
```

### التطبيق لا يبدأ
- امسح بيانات التطبيق: الإعدادات > التطبيقات > تطبيق مراقبة الطفل > حذف البيانات
- أعد تثبيت التطبيق

## نصائح مهمة

1. **الاتصال بالإنترنت**: تأكد من أن كلا الهاتفين متصلان بالإنترنت
2. **الخادم**: تأكد من أن الخادم يعمل وقابل للوصول
3. **الصلاحيات**: تأكد من تفعيل جميع الصلاحيات المطلوبة
4. **البطارية**: فعّل وضع البطارية الممتدة لضمان استمرار المراقبة

## الدعم والمساعدة

إذا واجهت أي مشاكل:
1. تحقق من الأخطاء في Logcat
2. تأكد من تثبيت جميع المتطلبات
3. جرّب حذف ملف `build` وإعادة البناء
4. تواصل مع الدعم الفني
