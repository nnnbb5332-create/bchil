# مراقب الطفل - Child Monitor App

تطبيق أندرويد متكامل لمراقبة نشاط الأطفال على هواتفهم الذكية بشكل آمن وفعال.

## المميزات الرئيسية

### نمط ولي الأمر
- ✅ **تسجيل حساب حقيقي** - يجب إنشاء حساب ببريد إلكتروني وكلمة مرور
- ✅ **تسجيل دخول آمن** - التحقق من البريد وكلمة المرور المشفرة
- ✅ **إضافة أطفال متعددين** - كل طفل له رمز فريد
- ✅ **عرض قائمة الأطفال** - مع أسمائهم ورموزهم
- ✅ **حذف الأطفال** - مع حذف جميع بياناتهم
- ✅ **عرض المواقع** - سجل المواقع المسجلة للطفل

### نمط الطفل
- ✅ **دخول بالرمز فقط** - لا يمكن الدخول إلا بالرمز الصحيح
- ✅ **خدمة خلفية** - لجمع البيانات تلقائياً
- ✅ **لا واجهة مرئية** - بعد تسجيل الدخول يعمل في الخلفية

## قاعدة البيانات
التطبيق يستخدم **SQLite** محلياً مع:
- تشفير كلمات المرور باستخدام **BCrypt**
- رموز فريدة للأطفال (6 أحرف)
- جداول: Parents, Children, Locations, AppUsage

## المتطلبات

- Android 12 (API 31) أو أحدث
- جهاز بـ 2GB RAM على الأقل
- اتصال إنترنت (اختياري للمزامنة)

## كيفية الاستخدام

### 1. تسجيل حساب ولي أمر
1. افتح التطبيق واختر "دخول ولي الأمر"
2. اضغط "إنشاء حساب جديد"
3. أدخل اسمك، بريدك الإلكتروني، وكلمة المرور
4. اضغط "إنشاء الحساب"

### 2. إضافة طفل
1. من لوحة التحكم، اضغط "إضافة طفل جديد"
2. أدخل اسم الطفل
3. سيتم إنشاء رمز فريد تلقائياً
4. انسخ الرمز واعطه للطفل

### 3. دخول الطفل
1. افتح التطبيق واختر "دخول الطفل"
2. أدخل الرمز (6 أحرف)
3. سيبدأ التطبيق في المراقبة تلقائياً

## البناء باستخدام GitHub Actions

### الطريقة 1: GitHub Actions (الأسهل)
1. ارفع المشروع إلى GitHub
2. GitHub Actions سيبني التطبيق تلقائياً
3. ستجد ملف APK في قسم Actions > Artifacts

### الطريقة 2: Android Studio
1. افتح المشروع في Android Studio
2. انتظر حتى ينتهي Gradle من البناء
3. اذهب إلى Build > Build Bundle(s) / APK(s) > Build APK(s)

### الطريقة 3: سطر الأوامر
```bash
./gradlew assembleDebug    # لبناء Debug APK
./gradlew assembleRelease  # لبناء Release APK
```

## الهيكل التقني

```
app/src/main/java/com/example/childmonitor/
├── activities/
│   ├── SplashActivity.kt              # شاشة البداية
│   ├── ParentLoginActivity.kt         # دخول ولي الأمر
│   ├── ParentRegisterActivity.kt      # تسجيل حساب جديد
│   ├── ParentDashboardActivity.kt     # لوحة تحكم ولي الأمر
│   ├── ChildLoginActivity.kt          # دخول الطفل بالرمز
│   └── AddChildActivity.kt            # إضافة طفل جديد
├── database/
│   └── DatabaseHelper.kt              # قاعدة البيانات SQLite
├── services/
│   ├── MonitoringService.kt           # خدمة المراقبة
│   └── AccessibilityMonitoringService.kt
├── monitoring/
│   ├── LocationMonitor.kt             # مراقبة الموقع
│   ├── AppUsageMonitor.kt             # مراقبة استخدام التطبيقات
│   └── BatteryMonitor.kt              # مراقبة البطارية
└── receivers/
    ├── CallReceiver.kt                # استقبال المكالمات
    └── SmsReceiver.kt                 # استقبال الرسائل
```
