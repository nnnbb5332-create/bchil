# تطبيق مراقبة الطفل - Child Monitor App

تطبيق أندرويد متكامل لمراقبة نشاط الأطفال على هواتفهم الذكية بشكل آمن وفعال.

## المميزات الرئيسية

### نمط ولي الأمر
- تسجيل دخول آمن ببريد إلكتروني وكلمة مرور
- إضافة أطفال متعددين مع كلمات مرور فريدة
- لوحة تحكم شاملة لمراقبة جميع الأطفال
- عرض الموقع الجغرافي المباشر على الخريطة
- مراقبة مستوى البطارية والشحن
- إحصائيات استخدام التطبيقات
- عرض المكالمات والرسائل والواتساب

### نمط الطفل
- تسجيل دخول برقم سري فقط
- خدمة خلفية لجمع البيانات
- لا توجد واجهة مرئية بعد التسجيل

## المتطلبات

- Android 12 (API 31) أو أحدث
- جهاز بـ 2GB RAM على الأقل
- اتصال إنترنت مستمر

## التثبيت

### الطريقة 1: استخدام Android Studio

1. فتح Android Studio
2. اختيار `File > Open` واختيار مجلد المشروع
3. الانتظار حتى ينتهي Gradle من البناء
4. الضغط على `Run > Run 'app'`

### الطريقة 2: بناء ملف APK

1. فتح Terminal في مجلد المشروع
2. تنفيذ الأمر:
```bash
./gradlew assembleRelease
```
3. سيتم حفظ الملف في: `app/release/app-release.apk`

## الصلاحيات المطلوبة

التطبيق يتطلب الصلاحيات التالية:

- **الموقع الجغرافي**: لتتبع موقع الطفل
- **المكالمات**: لمراقبة سجل المكالمات
- **الرسائل**: لمراقبة الرسائل النصية
- **الكاميرا والمايكروفون**: لالتقاط الصور والفيديو
- **الإنترنت**: لإرسال البيانات إلى الخادم
- **خدمات الموقع**: لتحديد الموقع بدقة

## الإعدادات المتقدمة

### تفعيل Accessibility Service

1. الذهاب إلى الإعدادات > إمكانية الوصول
2. اختيار "خدمات إمكانية الوصول"
3. البحث عن "تطبيق مراقبة الطفل"
4. تفعيل الخدمة

### تفعيل Device Admin

1. الذهاب إلى الإعدادات > الأمان
2. اختيار "مسؤولو الجهاز"
3. تفعيل "تطبيق مراقبة الطفل"

### إعدادات البطارية (Android 12+)

1. الذهاب إلى الإعدادات > البطارية
2. اختيار "تحسين البطارية"
3. اختيار "تطبيق مراقبة الطفل"
4. اختيار "غير محدود"

## الملفات الرئيسية

### Activities
- `SplashActivity`: شاشة البداية
- `ParentLoginActivity`: تسجيل دخول ولي الأمر
- `ChildLoginActivity`: تسجيل دخول الطفل
- `ParentDashboardActivity`: لوحة التحكم
- `AddChildActivity`: إضافة طفل جديد

### Services
- `MonitoringService`: الخدمة الرئيسية للمراقبة
- `AccessibilityMonitoringService`: خدمة إمكانية الوصول

### Monitoring
- `LocationMonitor`: مراقبة الموقع الجغرافي
- `AppUsageMonitor`: مراقبة استخدام التطبيقات
- `BatteryMonitor`: مراقبة البطارية

### Database
- `DatabaseHelper`: مساعد قاعدة البيانات

### API
- `ApiClient`: عميل API للتواصل مع الخادم

## الاتصال بالخادم

التطبيق يتصل بالخادم عبر:
- **Base URL**: `https://your-server.com/api/trpc`
- **Protocol**: HTTP/HTTPS
- **Format**: JSON

## الأمان

- جميع كلمات المرور مشفرة
- البيانات تُرسل عبر HTTPS
- لا يتم حفظ كلمات المرور بشكل واضح
- جميع البيانات الحساسة محمية

## استكشاف الأخطاء

### التطبيق لا يبدأ
- تحقق من أن لديك Android 12 أو أحدث
- امسح ذاكرة التخزين المؤقت: `./gradlew clean`

### لا يوجد اتصال بالخادم
- تحقق من الاتصال بالإنترنت
- تأكد من عنوان الخادم في `ApiClient.kt`

### الموقع لا يُحدّث
- تفعّل خدمات الموقع
- امنح التطبيق صلاحية الموقع
